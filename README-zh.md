# Arduino LED 亮度控制專案

## 專案描述

這個專案是為了學習**按鈕除彈跳**與**類比輸出**功能而設計的 Arduino 練習專案。

## 功能特色

- 使用按鈕控制 LED 亮度
- 實作按鈕除彈跳機制，避免誤觸發
- 透過 PWM (類比輸出) 控制 LED 亮度
- 5 段式亮度調節：0, 64, 128, 192, 255

## 接線說明

- **LED 正極**：連接到 Pin 9 (PWM 輸出腳位)
- **按鈕**：連接到 Pin 2 (使用內建上拉電阻)
- **LED 負極**：透過電阻連接到 GND
- **按鈕另一端**：連接到 GND

## 工作原理

1. **按鈕除彈跳**：使用時間間隔檢測，避免按鈕機械彈跳造成的多次觸發
2. **類比輸出**：使用 `analogWrite()` 函數產生 PWM 訊號控制 LED 亮度
3. **狀態循環**：每次按下按鈕，亮度增加一個階段，到達最大值後重新從 0 開始

## 學習重點

- 數位輸入處理與除彈跳技術
- PWM 類比輸出原理與應用
- Arduino 基本 I/O 操作
- 狀態機程式設計概念

## 使用方法

1. 按下按鈕，LED 亮度會逐漸增加
2. 共有 4 個亮度等級可循環切換
3. 達到最大亮度後，下次按鈕會回到最低亮度

## 技術參數

- 除彈跳延遲：25 毫秒
- 亮度階段：5 級 (0, 64, 128, 192)
- PWM 頻率：Arduino 預設 (約 490Hz)
